# Agentic chatbot

## This is a safety-focused chatbot application designed to assist users in staying safe by providing guidance, support, and real-time information.

The app combines a Flutter frontend for a user-friendly mobile experience with a powerful AI-powered backend that understands user queries, identifies intent, and responds empathetically. It can provide practical assistance such as locating nearby hospitals, police stations, or safe spaces, and helps users report incidents when needed.
The AI uses a Retrieval-Augmented Generation (RAG) approach with a structured knowledge base to deliver accurate safety information. Overall, the app integrates technology and research-based design to create a reliable digital safety companion.For more details about the backend architecture, and tools, please refer to the README in the backend folder.
